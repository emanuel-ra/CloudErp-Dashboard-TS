## Run Locally 💻
Clone the project  

~~~bash  
  git clone git@github.com:emanuel-ra/CloudErp-Dashboard-TS.git
~~~

Go to the project directory  

~~~bash  
  cd [project folder]
~~~

Install dependencies  

~~~bash  
pnpm install
~~~

Start the server  

~~~bash  
pnpm run dev
~~~
